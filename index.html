뷰티넷 백오피스 가이드

백오피스란 뷰티넷을 관리하기 위한 관리자 사이트입니다.

주의!백오피스에 접근하기 위해서는 다음과 같은 사항들이 필요합니다.

  • 방화벽 신청
  • 백오피스 계정 신청

1. 방화벽 신청

백오피스는 모든 사용자에게 접근을 허용하지 않기에 별도의 방화벽 신청이 필요합니다.

1) 그룹웨어 전자결재 > 방화벽 신청

전자결재 > 전산 > 방화벽정책등록/변경 신청서

2) 개인정보 입력

① 항목에 맞게 빈칸을 모두 채워 줍니다.

② 사용자 IP의 경우 개인 IP를 작성 해주시기 바랍니다.

③ 백오피스방화벽 신청서의 경우 어플리케이션 정책란은 공란으로 두셔야 됩니다.

④ 결제라인 : 각 팀장 // 접수자 : 인포테크실 김철환 차장

3) 신청할 방화벽 IP
