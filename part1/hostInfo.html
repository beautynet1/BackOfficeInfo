뷰티넷 백오피스 가이드

백오피스란 뷰티넷을 관리하기 위한 관리자 사이트입니다.

1. 뷰티넷 서버 구성

1) 서버 구성

뷰티넷은 개발서버, 스테이지서버, 운영서버 크게 3개의 형태로 되어 있습니다.

① 운영서버

= 일반 사용자들에게 노출되는 서버 (실서버)

② 스테이지 서버

= 일반 사용자들에게 노출이 되지 않는 테스트 서버로, 운영서버에 반영 되기 전단계의 서버 (운영 DB 사용)

③ 개발 서버

일반 사용자들에게 노출되지 않는 테스트 서버로, 스테이지 서버에 반영 되기 전단계의 서버 (개발 DB 사용)

2) 서버 반영 단계

※ 개발 서버 > 스테이지 서버 > 운영서버 순으로 개발된 내용이 반영됩니다.

2. 호스트 변경

1) 호스트 변경이란?

개발서버, 스테이지서버에 접근 하기 위해서는 호스트 변경이 필요합니다.

2) 호스트 변경 방법

1번 방법으로 호스트 변경이 안될시, 2번 방법을 통해 수동 변경 하셔야 합니다.


  • 1. 배치 파일을 통한 자동변경
  • 2. 수동 변경
파일을 다운 받은 후 압축을 해제 하고 나면 다음과 같이 보입니다.




[변경 후 화면]



배치파일을 이용한 호스트 변경이 안될 시 수동 변경을 합니다.


변경 방법

1) C:\Windows\System32\drivers\etc 폴더 접근

2) hosts 파일 클릭 (메모장 열기)

3) 다운 받은 파일 에서 원하는 서버 정보 골라 hosts 파일에 붙여 넣기 한 후 저장

[예제]

4) 모든 인터넷 창 닫고, 재실행

5) 변경 여부 위와 동일하게 확인
